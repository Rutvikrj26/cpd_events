## Product Overview

CPD Events is a comprehensive platform for:
- **Event Management**: Create and manage CPD events
- **Registration & Payments**: Handle event registrations and payment processing
- **Certificate Generation**: Automated certificate creation and delivery
- **Learning Management**: Course catalog and learning paths
- **Integrations**: Zoom, Stripe, and other third-party services
